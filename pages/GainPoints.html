Collecting Points

After a spark has lost it's charge, it's time to collect points and reset the board for the next player's turn.

Memory Slot

Each player collects a Victory Point for each active memory slot they have a player token on.

Capictor Slot

Each player collects a Victory Point for each capictor slot they have a player token and a point token on. Remove the point token but leave the player token.

Collect Point Tokens

Collect all other point tokens on the board and add those points to current player's Victory Point tracker.

Circuit Board Reset

The current player removes their spark and all battery tokens are discarded on the board and with the current player. Cards are drawn into the supply until the supply has as many cards as players plus one extra. If the draw pile is empty, nothing happens.

End of Round

When the last player has finished their turn, the round ends. The current event card is discarded and the next one in the stack is flipped over.

The first player will then begin the next round.

End of Game

The game ends when the third round is complete. The player who has the most victory points wins! If there is a tie, whoever has the most active capictors of the tie players win! If there is still a tie, count active memory slots between the tie players whoever has more wins!
